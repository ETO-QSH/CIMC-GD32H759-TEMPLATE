# Program：TicTacToe_byETO

## 程序简介

- 工程名称：tripleT
- 实验平台：CIMC IHD V0.4
- MDK版本：5.25

## 板载资源

 - GD32H759IMK6 MCU

│  ├─I2C
│  ├─KEY
│  ├─LED
│  ├─OLED
│  └─USART
├─HeaderFiles		头文件
├─Library			库文件
│  ├─GD32F4xx_standard_peripheral
│  └─Utilites/gd32h759i_eval
├─Middleware		中间件
│  └─TFLiteMicro
├─Model			模型数组
├─project		工程文件
├─Python		训练代码
├─Startup		启动文件
├─User
└─Readme		工程项目说明
